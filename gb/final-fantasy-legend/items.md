# Items

{% hint style="info" %}
Remember that all values are inserted mirrored into the save file, for example:

`05` will be inserted into the save file as `05 50`
{% endhint %}

| Hex Value | Item                                                       |
| --------- | ---------------------------------------------------------- |
| 00        | Potion                                                     |
| 01        | XPotion                                                    |
| 02        | Cure Book                                                  |
| 03        | Rod                                                        |
| 04        | Needle                                                     |
| 05        | Symbol                                                     |
| 06        | Eyedrop                                                    |
| 07        | Revive                                                     |
| 08        | Elixir                                                     |
| 09        | Heart (the symbol, i.e. the item to allows another revive) |
| 0A        | Arcane                                                     |
| 0B        | Door                                                       |
| 0C        | Strong                                                     |
| 0D        | Agility                                                    |
| 0E        | HP200                                                      |
| 0F        | HP400                                                      |
